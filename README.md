## ðŸš¦ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Krishprajapati15/QR-Generator.git
   cd QR-Generator
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
